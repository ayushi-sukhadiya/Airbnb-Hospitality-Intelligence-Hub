Airbnb SQL Analysis Project

This project contains SQL queries for exploring and analyzing an Airbnb dataset. It focuses on extracting insights from two main tables: listings and booking_details.

🗂️ Files
Airbnb.sql – Contains all SQL queries used for analysis

🛠️ Tools Used
Dbeaver-SQL (or any compatible SQL database)

🚀 How to Use
Import the SQL file into your database
Run the queries step by step
Analyze the outputs for insights
